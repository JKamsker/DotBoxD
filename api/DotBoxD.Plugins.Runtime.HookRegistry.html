FireAsync<TContext, TResult>(TContext, CancellationToken)

Dispatches the result-returning hooks (.Register(...) / .RegisterLocal(...)) installed for TContext in descending priority order, returning the first successful TResult or null when none is registered or none succeeds. The host applies the returned result to its live state. The result type is named explicitly here because the host — unlike the plugin authoring side, where it is inferred from the [Hook] context — already knows the type it will apply.
