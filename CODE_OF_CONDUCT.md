# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1,
as its Code of Conduct.

In short: we are committed to providing a welcoming, harassment-free experience for everyone.
Be respectful, assume good faith, and help keep discussions constructive and on-topic.

The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.

## Enforcement

Concerns about conduct may be reported privately to the maintainer (`@JKamsker`) via the contact
methods on their GitHub profile. Reports will be reviewed and handled confidentially.
