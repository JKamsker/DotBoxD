Tutorials

Three end-to-end walkthroughs, one per mode. Each starts from an empty project and ends with something you can run.

  1. First Service (RPC) — define a [DotBoxDService] contract, host it, and call it from a client over a typed proxy.
  2. First Kernel (sandbox) — import validated JSON IR and execute it inside a fuel-metered sandbox with a hard resource budget.
  3. Pushdown server extension — ship a plugin-supplied, server-side batch operation that collapses N remote calls into one.

New to the project? Read Getting started first, then come back here. For the concepts behind each mode, see the Guide.
